# 云顶尖锐化方案

## 问题
默认渲染下云边总是模糊，无法做出积雨云顶那种锐利清晰的轮廓。

根因（按影响排序）：
1. **缓存低通**：质量模式 0/1 把密度烘焙进 `densityRes³`（默认 96³）3D 纹理，三线性插值把比体素（≈0.09 世界单位）更细的硬边抹成渐变。
2. **软不透明度传递**：积分用 `1-exp(-d·step)`，边缘 `d` 小 → alpha 低 → 半透明渐隐。真实积雨云顶是含水量近阶跃突变、瞬间饱和。
3. **刻意圆化**：`vEnvelope = pow(vT, 2)*0.55` 专门把顶/底渐隐成圆顶，与尖锐顶相反。
4. **细节噪声只加绒毛**：模式 1 的乘性 detail 噪声制造 wispy 毛边，不增锐度。

## 核心思路
锐化放在 **raymarch 采样时按密度做**，不靠堆缓存分辨率（立方增长、不划算）。逐采样运算天然全屏分辨率，不受体素低通限制。

给"积雨云"这类需要硬顶的 preset 加一个 per-preset 参数 `edgeHardness`（锐度），驱动下面三步。

## 落地步骤

### 1. 陡峭密度传递函数（解决"软"，≈零成本）
采样到密度 `d` 后过窄窗：`smoothstep(thr-w, thr+w, d)`，窗宽 `w` 由 `edgeHardness` 控制。锐度高 → 窗口极窄 → alpha 一两步内从 0 到饱和。配合调高 `densityScale` 让 `1-exp(-d·step)` 快速饱和。约拿到 70% 效果。

### 2. 边界高频解析侵蚀（解决"锐但发糊/发飘"，成本中等）
仅在边缘带（`d` 接近 `thr`）叠加一层高频 worley/curl 噪声去"啃"边缘。
关键：该噪声在 raymarch 内**解析计算、不进缓存**，绕开三线性插值，给出积雨云"花椰菜"硬块感。这是质量关键步。

### 3. 去掉云顶圆化（解决"顶部是圆的"，≈零成本）
对硬顶类型，把顶部一侧的 `vEnvelope` 换成接近硬截断（或做成砧状 anvil 上沿），让云顶贴噪声等值面而非被渐隐。

### 配套：缓存
做完第 2 步后可见边缘主要由解析高频决定，缓存只管团块主体，分辨率可不动。若主体边缘仍有波纹，再把这类 body 局部走质量模式 2（解析、不缓存），不必全局提分辨率。

## 取舍与顺序
- 第 1+3 步：几乎免费，先做。
- 第 2 步：质量关键，成本中等。
- 模式 2：最锐最贵，可选。
- 建议顺序：1 → 3 → 2，按 preset 暴露 `edgeHardness` 滑杆；普通积云保持现状，积雨云调高。

## 涉及文件
- `shaders/cloud.wgsl`：传递函数、边界侵蚀、vEnvelope、densityAt/fs。
- `src/params.ts`：新增 `edgeHardness`（per-preset，打包进 PresetShape 或 Globals）。
- `src/gui.ts`：暴露滑杆。
